# Countdown WC PWA

A countdown progressive web app that uses web component.

## Features

- [x] do countdown given a timer name, target date and time
- [x] store timers in localstorage
- [x] add timer
- [x] remove timer
- [x] work offline
- [x] edit timer
- [ ] notification
- [ ] a more acceptable UI
